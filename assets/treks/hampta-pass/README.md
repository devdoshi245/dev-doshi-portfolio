# Hampta Pass gallery — expected files

Upload your Hampta Pass photos to this folder (GitHub → branch `trek-photos` →
Add file → Upload files) using these exact names:

```
01.jpg  02.jpg  03.jpg  04.jpg  05.jpg  06.jpg  07.jpg  08.jpg
```

- Use as many as you have (minimum ~4 looks good; up to 8 wired by default).
- `01.jpg` should be your favorite — once wired, the gallery shows them in order.
- The card cover can stay the current one or switch to any photo — say which
  when you upload.
- Export tip: ~1600px long edge, JPEG quality ~80. If your phone exports
  `.jpeg`/double extensions, don't worry — they'll be normalized before merge.
